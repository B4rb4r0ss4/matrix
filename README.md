# matrix
Hi, there is my matrix implementaion in C++ and SFML

You can run it: 
```
git clone https://github.com/B4rb4r0ss4/matrix
cd matrix
make
./main
```
usage option: 
```
IN PROGRAM 
    A - increase speed
    Z - decrease speed
```
